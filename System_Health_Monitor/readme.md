# 🖥️ System Health Monitor

## 📌 Overview
This script monitors basic system health metrics and writes them to a log file.  
It checks:
- CPU usage
- Memory usage
- Disk usage
- Top 5 processes by CPU
- System uptime

## ⚙️ Usage
```bash
chmod +x sys_health_monitor.sh
./sys_health_monitor.sh
cat health.txt
